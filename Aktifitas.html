Aktifitas Dalam sehari

bangun tidur saya langsung mandi dan sarapan setelah itu saya berangkat ke kampus.

dikarenakan jarak kampus dan rumah saya membutuhkan waktu 1 jam an maka saya harus bersiap siap agar tidak terlambat.

setelah mata kuiah selesai saya biasanya mampir ke kost temen untuk bermain atau numpang istirahat.

setelah beristirahat saya otw pulang karna rumah saya jauh kadang di jalan kalau hujan ataupun macet bisa bikin tambah lelah.

karna lelah biasanya sampai rumah saya bermain dengan adek saya ataupun langsung beristirahat.

karna saya punya pacar semua proses itu saya jalani dengan pacar saya.
